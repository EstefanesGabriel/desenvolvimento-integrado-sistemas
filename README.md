Configurações para o Servidor DotNet:
Os arquivos de matriz devem estar com os seguintes nomes:
 - H-1.csv para a matriz 60x60
 - H-2.csv para a matriz 30x30

Os arquivos de sinal G devem estar com os seguintes nomes:
 - g-60x60-1.csv
 - g-60x60-2.csv
 - g-60x60-3.csv
 - g-30x30-1.csv
 - g-30x30-2.csv
 - g-30x30-3.csv
